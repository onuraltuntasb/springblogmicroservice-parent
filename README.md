## About The Project
springblog is basicaly a poc project to see how to develop microservices in
spring boot ecosystem with docker and kafka

The three principles of project:

- Clean templating for future projects
- Dockerise microservices and databases
- Learn simple usage of webclient api between microservices

## Getting Started

### Prerequisites

-   JDK 17 or upper
-   Maven

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/onuraltuntasb/springblogmicroservice-parent
    ```
2. Run services
 ```sh
     mvn spring-boot:run
   ```
